# Telepítés és futtatás

## 1. Klónozd vagy töltsd le a tárolót:
```bash
git clone https://github.com/balazsNorbert/Termek-Feladat.git
```
## 2. Lépj be a projekt mappájába:
```bash
cd termek-feladat
```
## 3. Indítsd el a beépített PHP szervert:
```bash
php -S localhost:8000
```
## 4. Nyisd meg a böngészőben:
http://localhost:8000
